WEBSHOP

KLIK BUITEN HET VAK OM TERUG TE GAAN

PROJECT DESCRIPTIE

Ik heb een webwinkel ontwikkeld met behulp van WebStorm en mijn kennis van HTML, CSS en JavaScript. De webwinkel heeft een aantrekkelijke en gebruiksvriendelijke interface en biedt aantal producten met gedetailleerde beschrijvingen en afbeeldingen. JavaScript werd hier gebruikt voor productfiltering, dynamisch winkelwagenbeheer en een soepel afrekenproces. Klanten kunnen eenvoudig bladeren, items aan hun winkelwagen toevoegen en aankopen voltooien. De combinatie van HTML, CSS en JavaScript resulteert in een volledig functionele en boeiende webwinkel.

WEBSITE VOOR ADRENALINE VERSLAVING

KLIK BUITEN HET VAK OM TERUG TE GAAN

PROJECT DESCRIPTIE

Ik heb een website gemaakt in het Nederlands met een uniek doel: mensen ondersteunen die problemen hebben met adrenalineverslaving. Dit is een van mijn eerste projecten op het gebied van webontwikkeling, waarbij ik begon te experimenteren met CSS. Dit platform is bedoeld om mensen te helpen los te komen van adrenalineverslaving. Het is een weerspiegeling van mijn beginnende reis in webontwikkeling en mijn inzet om een positieve impact te hebben. Vanaf de eerste dagen van het leren van CSS tot het creëren van een hulpmiddel dat het leven van mensen wil verbeteren, laat dit project mijn streven naar voortdurende verbetering zien en het vermogen van technologie om positieve verandering te brengen.

GAME DEVELOPMENT

KLIK BUITEN HET VAK OM TERUG TE GAAN

PROJECT DESCRIPTIE

In deze periode heb ik gewerkt aan C# en game-ontwikkeling. Mijn verkenning van game-ontwikkeling is zowel dynamisch als samenwerkend geweest. Vaardig in zowel 2D als 3D domeinen binnen Unity, heb ik succesvol een meeslepende 2D shooter game gemaakt, onafhankelijk, en samengewerkt aan een 3D shooter met een team van vier ontwerpers en een mede-programmeur. Deze samenwerkingsinspanning heeft niet alleen mijn technische vaardigheden aangescherpt, maar benadrukte ook het belang van teamwork bij het leveren van meeslepende game-ervaringen.

In het domein van softwareontwikkeling heb ik een programma ontworpen en geïmplementeerd voor het aanmaken van tickets met behulp van Visual Studio, waarbij ik mijn vaardige begrip van C# heb laten zien. Deze ervaring benadrukt mijn vermogen om programmeerkennis om te zetten in praktische toepassingen.

Bovendien heb ik mijn vaardigheid in animatie aangetoond door boeiende animaties te creëren in Unity. Ik heb mijn vaardigheden in code-structuur verfijnd door te werken met klassen in Visual Studio en het implementeren van overerving in Unity. Deze aspecten hebben niet alleen de visuele aantrekkingskracht van mijn projecten verbeterd, maar ook mijn begrip van fundamentele programmeerprincipes versterkt.

Mijn ervaring in het samenwerken aan Unity-projecten met een andere ontwikkelaar, waarbij we GitHub hebben gebruikt voor versiebeheer, illustreert mijn aanpassingsvermogen en effectiviteit in teamomgevingen. Deze vaardigheid garandeert gestroomlijnde samenwerking, efficiënte projectontwikkeling en effectief versiebeheer.
